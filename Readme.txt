Dataset for the paper "CV Aided Beamforming Fused with Limited Feedback".

Unzip dataset.zip.
Folder "pics" contains the captured images.
Folder "weights" contains the normalized channel coefficients for IQ channels of 4 antennas with dimension (4, 2). Use eval() function to obtain variables.
Folder "positions" contains the 2D position (x,y) of the UE. Use eval() function to obtain variables.
